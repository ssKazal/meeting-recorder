Smart Meeting Recorder

APT repository for Debian / Ubuntu

Detects when you join an online meeting and asks before recording your screen, microphone and system audio. Stops automatically when the call ends.

Install

curl -fsSL __PAGES_URL__/KEY.gpg | sudo gpg --dearmor -o /usr/share/keyrings/meeting-recorder.gpg

echo "deb [arch=all signed-by=/usr/share/keyrings/meeting-recorder.gpg] __PAGES_URL__ stable main" \
  | sudo tee /etc/apt/sources.list.d/meeting-recorder.list

sudo apt update
sudo apt install meeting-recorder

Updates then arrive through apt upgrade like any other package.

Use

meeting-recorder start      # start the background service
meeting-recorder status     # service state + detected meeting
meeting-recorder settings   # settings window

⚠️ X11 only. Screen capture uses x11grab; Wayland is not supported yet. Check with echo $XDG_SESSION_TYPE — it must print x11. If it prints wayland, log out and pick “Ubuntu on Xorg” at the login screen.

Uninstall

sudo apt remove meeting-recorder
sudo rm /etc/apt/sources.list.d/meeting-recorder.list /usr/share/keyrings/meeting-recorder.gpg
